Applets

Java Applets Overview

  • An applet in Java is essentially a Java program that is embedded within a webpage.
  • Functionality of Java applets includes:
    1. Playing music.
    2. Playing videos.
    3. Creating animations.
    4. Displaying graphics.
    5. Handling user data input.
    6. Retrieving data from a database.
    7. Primarily used to generate dynamic content at the client side.
  • Key points about Java applets:
    1. Extends the java.applet.Applet class.
    2. Does not contain a main() method.
    3. Life cycle methods include:
      • init()
      • start()
      • paint() (this method is present in the java.awt - Abstract Window Toolkit - Component class)
      • stop()
      • destroy()
    4. Applets do not use the System.out.println() method.
    5. To execute an applet, you can use the appletviewer tool or a web browser. The appletviewer tool is part of the JDK. If using a browser, additional plugins are required for proper applet execution.

Applet Life Cycle methods in Java

init Method

                    
public void init()
  • The init method is crucial for initializing the applet, allowing tasks such as variable initialization, setting background and foreground colors, and allocating memory for controls (e.g., labels, checkboxes).
  • This method is executed only once during the entire life cycle of the applet.
  • When the applet is first loaded into memory, the init method is called.

start Method

                    
public void start()
  • The start method is executed after the init method.
  • Useful for restarting an applet if its execution has been stopped.
  • Restarting the execution of the current webpage is achieved using the start method.
  • There's no predictable limit to how many times an applet webpage can be restarted; it can be restarted multiple times.
  • Whenever switching back to the same page, the start method will be executed again.
  • When we click on the maximize button than this method can be called.

paint Method

                    
public void paint(Graphics g)
  • The paint method is used to display content in the view area of the applet.
  • With the Graphics class, one can paint shapes like circles and rectangles, as well as display text within the applet.
  • This method is executed after the start method.

stop Method

                    
public void stop()
  • The stop method is employed to halt the execution of an applet.
  • Execution of the current webpage is stopped using the stop method when transitioning from one webpage to another. Additionally, clicking the minimize button on the applet window triggers the execution of the stop method.

destroy Method

                    
public void destroy()
  • The destroy method serves to release the applet object's memory. When this method is called, the applet memory is destroyed, making it available for use in other programs.
  • Clicking on the cross button of the applet window triggers the execution of the destroy method.
  • The destroy method is executed only once in the entire life cycle of an applet.
  • It's worth noting that the init and destroy methods are executed only once, whereas the remaining methods can be executed multiple times.
  • Remember: The init(), start(), stop(), and destroy() methods are present in the Applet class, which is part of the java.applet package.
