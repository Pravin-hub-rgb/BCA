Point Representation

Alright! Let's start with the basics. Imagine placing a dot on a piece of paper—this dot represents a point. But how do we describe where this point is? We use a coordinate system for that.

A point is a fundamental element that represents a position in space. It does not have a size, shape, or any other property except its location.

2D Coordinate System

In a 2D (two-dimensional) space, we use an X-axis (horizontal) and a Y-axis (vertical) to locate a point. A point is written as:

\[ P(x, y) \]

For example:

  • \( P(3, 5) \) means the point is at x = 3 and y = 5.
  • \( P(-2, 4) \) means the point is 2 units left and 4 units up.

In 2D graphics, every object you see—lines, shapes, images—is made up of points positioned at specific coordinates.

3D Coordinate System

Now, in 3D (three-dimensional) space, we add another axis: the Z-axis, which represents depth. A 3D point is written as:

\[ P(x, y, z) \]

For example:

  • \( P(2, 3, 5) \) means the point is at \( x = 2 \), \( y = 3 \), and \( z = 5 \) (depth).
  • \( P(-1, 4, -3) \) means it is left by 1 unit, up by 4 units, and back by 3 units.

Just like in 2D, every 3D model you see in games or animations is made up of points connected in space.

Distance Between Two Points

How far is one point from another? We use the distance formula to find out.

In 2D, for two points \( (x_1, y_1) \) and \( (x_2, y_2) \):

\[ d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2} \]

Example:

  • Find the distance between \( (3, 4) \) and \( (6, 8) \):
\[ d = \sqrt{(6 - 3)^2 + (8 - 4)^2} = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5 \]

In 3D, the formula expands to include the \( z \)-coordinate:

\[ d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2} \]

Distance is important in graphics for detecting collisions, measuring object placement, and determining movement.

Midpoint Formula

Sometimes, we need to find the middle point between two given points. This is done using the midpoint formula.

In 2D, for two points \( (x_1, y_1) \) and \( (x_2, y_2) \), the midpoint is:

\[ M = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right) \]

Example:

  • Find the midpoint between \( (2, 6) \) and \( (8, 4) \):
\[ M = \left( \frac{2 + 8}{2}, \frac{6 + 4}{2} \right) = \left( \frac{10}{2}, \frac{10}{2} \right) = (5, 5) \]

In 3D, we just add \( z \):

\[ M = \left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2}, \frac{z_1 + z_2}{2} \right) \]

The midpoint helps in dividing objects, detecting symmetry, and creating smooth curves in graphics.
